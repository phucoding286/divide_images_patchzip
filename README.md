### Công cụ này giúp chia dữ liệu lớn của ảnh ra thành từng patch con, bao gồm shuffle, trộn ảnh, giúp phân phối đồng đều, hữu ích cho huấn luyện DDPM hoặc latent diffusion, hay generative models nói chung.
-
### Cách sử dụng
1. chạy file ```main.bat``` hoặc ```images_data_patching.py```.
2. sau đó nhập các folders path chứa ảnh bên trong vào.
3. cuối cùng nhập số lượng ảnh mong muốn trên mỗi batch file zip và chạy.
